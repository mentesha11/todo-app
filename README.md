# TODO - Todo react app

Visit my live portfolio: [mytodonoteweb.netlify.app](https://mytodonoteweb.netlify.app)

## Tech Stack

- **Frontend**: React.js  
- **Hosting**: Netlify  

## Run Locally

1. Clone the repo:  
   `git clone https://github.com/mentesha11/mytodonoteweb.git`
2. Install dependencies:  
   `npm install`
3. Start the server:  
   `npm start`
